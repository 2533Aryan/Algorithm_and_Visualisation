K-Nearest Neighbor (KNN)

Javascript Library Used: d3ML.

Click on the dark area to create new circles and see how their color changes based on their neighbors.

In the visualization you can see how the k-nearest neighbor (KNN) algorithm classifies new data points based on their k nearest neighbors, where the amount of neighbors k is specified using the slider above. The new point will be assigned the class that the majority of its k nearest neighbors hold.

If you check the "Weighted" checkbox above, the algorithm is justified, so that the inverse of the distance of the k neighbors is taken into account. This is important during "ties", meaning that you chose an even amount of neighbors k and half of them are class red while the other half are blue.

If you don't use the weighted version of KNN in this case, the neighbor with the closest distance will ALWAYS win.
