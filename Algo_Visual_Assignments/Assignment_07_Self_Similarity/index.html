Barnsley Fern

The code uses recursion to generate the Barnsley fern fractal. Starting from the initial point (0, 0), the generatePoint function is called repeatedly with different transformation rules and generates a single point on the canvas. Each point is colored based on the depth of recursion that produced it. The function generateFractal uses a loop to generate the Barnsley fern fractal by repeatedly calling generatePoint with different initial points and colors based on recursion depth.

The code is based on the Barnsley fern fractal discovered by mathematician Michael Barnsley in 1988. The mathematical basis for the Barnsley fern can be found in his paper "Fractals Everywhere" (1988).
